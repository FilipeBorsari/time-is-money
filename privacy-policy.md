# Política de Privacidade

_Última atualização: 9 de abril de 2026_

A extensão **Tempo é Dinheiro** foi desenvolvida com respeito à privacidade do usuário. Esta política descreve quais dados são utilizados e como.

## Dados coletados

A extensão armazena localmente, via `chrome.storage.sync`, apenas:

- Salário mensal líquido informado pelo usuário
- Jornada de trabalho diária informada pelo usuário
- Estado de ativação da extensão (ativo/inativo)

Esses dados são de preenchimento voluntário, ficam armazenados na conta do Chrome do próprio usuário e são usados exclusivamente para calcular o tempo de trabalho equivalente aos preços exibidos nas páginas visitadas.

## Dados não coletados

A extensão **não**:

- Envia nenhum dado a servidores externos
- Coleta histórico de navegação, URLs ou conteúdo das páginas visitadas
- Utiliza análises, rastreamento ou publicidade
- Compartilha informações com terceiros

## Permissões utilizadas

- **storage**: para salvar as configurações do usuário
- **activeTab**: para exibir o badge de tempo de trabalho na aba ativa
- **host_permissions (`<all_urls>`)**: para detectar preços em qualquer site visitado

## Contato

Dúvidas sobre esta política podem ser enviadas para o desenvolvedor via página da extensão na Chrome Web Store.
